# Test Automation Framework

This is a comprehensive test automation framework designed to streamline UI, API and Database testing. It leverages Playwright and TypeScript to provide a powerful automation solution. It also automates businessflows touching up Azure Services like Azure Databricks. Ensure you have the following prerequisites installed before setting up the framework.

## Prerequisites

Before getting started, make sure you have the following software installed:

1. **Visual Studio Code (VS Code):** A lightweight and powerful code editor. You can download it from [here](https://code.visualstudio.com/).

2. **Node.js:** A JavaScript runtime for executing JavaScript on the server-side. You can download it from [here](https://nodejs.org/).

3. **Git:** Version control system for tracking changes in your codebase. You can download it from [here](https://git-scm.com/).
